# Setting development enviroment

1. Clone the repository
2. Execute
```
yarn install
```
3. download the nest cli if you do not have item-toggle
```
npm i -g @nestjs/cli
```
4. Up and run the database
```
docker-compose up -d
```

## Stack

* NestJs
* MongoDB
